Music Player Web Application

This project is a simple and interactive web-based music player that allows users to play, pause, and manage their favorite songs directly in the browser. It is designed with a clean user interface and smooth user experience in mind.

Overview

The Music Player application provides basic audio playback functionality using modern web technologies. It is lightweight, easy to use, and serves as a great beginner-to-intermediate level project for understanding frontend development.

Features
Play and pause songs
Next and previous track controls
Display of song title and artist
Progress bar to track playback
Responsive design for different screen sizes
Simple and clean UI
Technologies Used
HTML
CSS
JavaScript
